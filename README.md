# IDX Semantic Spine

A governed controlled vocabulary for the corporate digital estate, developed by IDX.

## What is the Semantic Spine?

The Semantic Spine is a classification framework of **107 node definitions** across **5 domains** that maps every section of a corporate website to a governed vocabulary with schema.org markup guidance.

### Domains

| Domain | Code | Nodes |
|--------|------|-------|
| ESG (Environmental, Social & Governance) | E | 27 |
| Investor Relations | I | 25 |
| Corporate Communications | C | 20 |
| Corporate Talent | T | 18 |
| Brands & Products | B | 17 |

## Purpose

The spine enables:
- **AI Engine Optimisation (AEO)** — structuring corporate content so AI models can retrieve, parse, and cite it accurately
- **Schema.org implementation** — each node maps to a specific schema.org type with JSON-LD examples
- **Content governance** — governed definitions, freshness thresholds, and AEO priority ratings for every content category
- **Site auditing** — classify any corporate URL against the spine to identify gaps and opportunities
